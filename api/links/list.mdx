---
title: 'List Links'
api: 'POST /links/list'
description: 'Returns all short links for an organization'
---

## Authentication

Requires API key with `read:links` scope.

## Request Body

<ParamField body="organizationId" type="string" required>
  Organization ID to list links for
</ParamField>

<ParamField body="externalId" type="string">
  Filter by external identifier
</ParamField>

## Response

Returns an array of link objects, ordered by creation date (newest first).

<ResponseField name="id" type="string">
  Link identifier
</ResponseField>

<ResponseField name="slug" type="string">
  URL slug
</ResponseField>

<ResponseField name="name" type="string">
  Display name
</ResponseField>

<ResponseField name="targetUrl" type="string">
  Destination URL
</ResponseField>

<ResponseField name="organizationId" type="string">
  Parent organization
</ResponseField>

<ResponseField name="createdBy" type="string">
  User who created the link
</ResponseField>

<ResponseField name="expiresAt" type="string | null">
  Expiration timestamp
</ResponseField>

<ResponseField name="ogTitle" type="string | null">
  Open Graph title
</ResponseField>

<ResponseField name="externalId" type="string | null">
  Your external identifier
</ResponseField>

<ResponseField name="createdAt" type="string">
  Creation timestamp
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Last update timestamp
</ResponseField>

## Example Request

```bash
curl -X POST https://api.databuddy.cc/links/list \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "organizationId": "org_abc123"
  }'
```
